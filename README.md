# Switch — переключатель раскладки

Десктопное приложение в стиле Punto Switcher: конвертирует текст, набранный не в той
раскладке, между английской (QWERTY) и русской (ЙЦУКЕН).

Пример: `ghbdtn? rfr ltkf&` → `привет, как дела?` (и обратно).

## Возможности

- **Глобальный режим** — работает поверх любой программы: выделите текст в браузере,
  мессенджере, редакторе и нажмите глобальную горячую клавишу — текст сменит раскладку
  прямо на месте. Работает только с выделенным текстом: приложение **не отслеживает и не
  накапливает набор** (это не кейлоггер) — глобальный хук лишь ловит одну заданную комбинацию.
- **Ручной режим** — окно с текстовым полем и кнопкой «Сменить раскладку»
  (без выделения — слово у курсора, с выделением — весь фрагмент).
- **Настраиваемая горячая клавиша** — по умолчанию **Ctrl+Shift+Space**, меняется в окне
  кнопкой «Изменить» (нажать нужную комбинацию). Сохраняется между запусками.
- **Размер шрифта** поля ввода настраивается и сохраняется между запусками.
- **Отдельное окно настроек** — горячая клавиша, размер шрифта, запуск свёрнутым и параметры
  обновлений вынесены из главного окна в «Настройки…».
- **Запуск свёрнутым** — по умолчанию приложение стартует сразу в трее (настраивается).
- **Обновление из программы** — окно «О программе» показывает версию и кнопку
  «Проверить обновления». По умолчанию новая версия скачивается и ставится автоматически
  (с перезапуском); режим можно переключить на «скачать в Загрузки». Опциональная тихая
  проверка при запуске (выключена по умолчанию) уведомляет, если вышла новая версия.
- **Один экземпляр** — повторный запуск не поднимает второе окно; работает ровно один
  экземпляр (блокировка файла `~/.switch/instance.lock`).
- **Сворачивание в трей** — рядом с часами (Windows) / в строке меню (macOS). Закрытие
  и сворачивание окна прячут его в трей, приложение продолжает работать фоном; выход —
  из меню трея.
- Автоопределение направления посимвольно: латиница → кириллица, кириллица → латиница.
- Кроссплатформенно: **Windows** и **macOS**. Один JAR содержит нативные библиотеки для
  обеих ОС.

### macOS: разрешение

Для глобального режима macOS требует выдать приложению доступ в
**Системные настройки → Конфиденциальность и безопасность → Универсальный доступ**
(Accessibility) — иначе оно не сможет читать и вставлять текст в другие программы.

## Требования

- JDK 21+
- Gradle Wrapper (входит в репозиторий)

## Запуск

Глобальная горячая клавиша по умолчанию — **Ctrl+Shift+Space** (меняется в окне).
В окне также есть кнопка **«Сменить раскладку»** для ручного режима.
