# Switch — переключатель раскладки

Десктопное приложение в стиле Punto Switcher: конвертирует текст, набранный не в той
раскладке, между английской (QWERTY) и русской (ЙЦУКЕН).

Пример: `ghbdtn? rfr ltkf&` → `привет, как дела?` (и обратно).

## Установка и запуск

1. Скачай `Switch.jar` из последнего релиза на вкладке
   [**Releases**](https://github.com/Kykyryky23/Switch-releases/releases).
2. Запусти:
   ```bash
   java -jar Switch.jar
   ```
   На **Windows** можно просто дважды кликнуть по `Switch.jar`.

Нужна установленная **JDK 21+**. Больше ничего ставить не надо — всё необходимое уже внутри JAR.
По умолчанию приложение стартует свёрнутым в трей (рядом с часами на Windows / в строке меню
на macOS).

## Возможности

- **Глобальный режим** — выдели текст в любой программе и нажми горячую клавишу
  (по умолчанию **Ctrl+Shift+Space**) — раскладка сменится прямо на месте.
- **Ручной режим** — окно с полем ввода и кнопкой «Сменить раскладку».
- **Настройки** (окно «Настройки…») — горячая клавиша, размер шрифта, запуск свёрнутым,
  параметры обновлений. Всё сохраняется между запусками.
- **Обновление из программы** — окно «О программе» показывает версию и кнопку
  «Проверить обновления»: новая версия скачивается, ставится и приложение перезапускается
  (можно переключить на «скачать в Загрузки»).
- **Один экземпляр** — повторный запуск не открывает второе окно.
- **Сворачивание в трей** — закрытие и сворачивание окна прячут его в трей; приложение
  продолжает работать фоном, выход — из меню трея.
- Работает на **Windows** и **macOS** (один JAR на обе ОС).

## Нюансы

- **macOS** — для глобального режима выдай приложению доступ в
  **Системные настройки → Конфиденциальность и безопасность → Универсальный доступ**
  (Accessibility), иначе оно не сможет читать и вставлять текст в другие программы.
- **Глобальный режим работает только с выделенным текстом** — приложение не отслеживает
  и не накапливает набор (это не кейлоггер), а лишь реагирует на одну заданную комбинацию.
- **Windows** — при автоматическом обновлении антивирус может спросить разрешение на
  перезапуск приложения; это нормально.
